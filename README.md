# Генерация текста на уровне символов с помощью RNN (Шекспир)

В этом проекте реализована рекуррентная нейронная сеть (RNN) на уровне символов с использованием TensorFlow/Keras для генерации текста в стиле Уильяма Шекспира.

Цель состоит в том, чтобы обучить модель на наборе данных произведений Шекспира, а затем использовать ее для генерации новых текстовых последовательностей, имитирующих его стиль, лексику и драматическую структуру.
